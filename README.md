# 🦠 Covid-19 in India — Tableau Dashboard Analysis

## 📌 Overview
This interactive dashboard provides a comprehensive visual analysis 
of the Covid-19 pandemic in India. It covers state-wise death rates, 
vaccination progress, testing statistics, and demographic breakdowns 
using real-world datasets.

## 📊 Dashboard Visualizations
- 🗺️ **Map** — Total Covid Deaths by Indian States
- 🍩 **Age Group Details** — Distribution of cases across age groups
- 📈 **Covid Cases by State** — Confirmed, Cured & Death trends over time
- 💉 **First vs Second Dose** — Vaccination dose comparison
- 👥 **Gender Distribution** — Male vs Female breakdown
- 🏥 **ICMR Testing Labs** — Number of labs per state
- 🧪 **Statewise Testing Details** — Negative, Positive & Total Samples
- 💊 **Doses by Vaccine** — CoviShield, Covaxin & Sputnik V comparison

## 🛠️ Tools & Technologies
- **Tableau Public** — Data visualization
- **Shapefile** — India State Boundary mapping
- **CSV Dataset** — Covid vaccine statewise data

## 📁 Dataset Used
- `covid_vaccine_statewise.csv` — State-wise vaccination records
- `India_State_Boundary.shp` — India map shapefile

## 🔍 Key Insights
- 20-29 age group had the highest Covid impact (24.86%)
- CoviShield was the most administered vaccine (61.9B+ doses)
- Uttar Pradesh had the highest number of total samples tested
- Maharashtra had the most ICMR testing labs (32)
- Male cases (72.24%) were significantly higher than female (27.76%)
